KTPlay Android SDK
===================

简介
-------------------
** KTPlay Android SDK V1.0.0 **

KTPlay SDK为游戏提供以下功能扩展

- 游戏内社区	玩家可以在游戏内交流
- 通知和奖励	提高用户粘性
- 交叉推广	推广你的游戏

目录结构
-------------------
- KTPlaySDK 		包含接入SDK所需的代码库和资源
- Docs 				包含SDK使用指南和其他文档
- Sample			包含SDK接入示例代码
- Plugins			包含三方平台插件

快速开始
-------------------
请查看Docs目录下的SDK使用指南
